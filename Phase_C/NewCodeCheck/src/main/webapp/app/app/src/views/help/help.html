    Common Questions

    How do I get started?
    Code Checker detects plagiarism between two JAVA programming assignments and returns a similarity metric. To get started, just create an account from the Sign Up page and get started detecting plagiarism

    How do I read the plagiarism report?
    The plagiarism report shows the two assignment submission files side-by-side for you to review manually, as well as displays a similarity percentage between the two files, which is calculated using our plagiarism detection algorithm.
    How do I detect plagiarism?
    Once you have the files you want to check, navigate to the Upload Submissions page, type in a name for your coding assignment, select the files from your computer, and click on the "Analyze Results" button and you are done!

    How does this really work?
    Code Checker analyzes the two uploaded files using a complex algorithm, and applies a number of transformations like standardizing variables names, method names, class and package names, the order of method definitions, removing comments, and unused imports, etc. and then compares the transformed files for similarities.
